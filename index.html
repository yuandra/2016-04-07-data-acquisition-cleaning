General Information

This website is for the data acquisition and cleaning training. The training will talk about how to get data using web and mobile based survey tools and how to clean up data using Open Refine. This workshop is for any researcher who want to learn how to get data and no prior computational experience is required. This workshop including the template of this website is based on Data Carpentry workshops.

We will cover {{page.day1_am}}, {{page.day1_pm}}, {{page.day2_am}} and {{page.day2_pm}}. Participants should bring their laptops and plan to participate actively. By the end of the workshop learners should be able to more effectively collect and clean data and be able to apply the tools and approaches directly to their ongoing research.

Who: The course is aimed at graduate students and other researchers.

Syllabus

Web survey using Google Forms

  • Introduction to Google Forms
  • Basic form creation using Google Form
  • Theming Google Forms survey
  • Data validation with Google Forms
  • Conditionality in Google Forms
  • Distributing Google Forms
  • Collaboration with Google Forms
  • Extending Google Forms

Mobile data collection using Kobo Toolbox

  • Introduction to Kobo Toolbox
  • Basic form creation using Kobo Toolbox
  • Data validation in Kobo Toolbox
  • Conditionality in Kobo Toolbox
  • Distributing & data collection using Kobo Toolbox
  • Collaboration in Kobo Toolbox
  • Multilanguage form in Kobo Toolbox

Introduction to Open Refine

  • Introduction to data cleaning
  • Setting up Open Refine
  • Project creation in Open Refine
  • Using Facet to select data
  • Basic data cleaning operation in Open Refine

Introduction to Open Refine (cont.)

  • Filtering in Open Refine
  • Splitting cells in Open Refine
  • Using clustering to find duplicates
  • Cell transposing using Open Refine
